# calculadora-juz-quran-thelightofsunnah
Este proyecto, titulado "Calculadora Juz’ Quran", ha sido desarrollado por TheLightOfSunnah – Abdel Bidan. Su objetivo es facilitar a los usuarios el cálculo del avance en la lectura del Qurán, permitiendo saber de forma sencilla cuántas páginas, ayat o juz’ faltan para completar la lectura diaria, semanal o mensual.
